Proyecto el Mirador.

Integrantes: Milton Saavedra Desarrollador/Tester, Marjorie Portiño Analista funcional.

En el edificio mirador las transacciones manuales y la lenta atencion a los arrendatarios a producido una enorme cantidad de morosidad, para combatir esto hemos
desarrollado un sistema de pago en linea que permite la paga y administracion de gastos comunes y extraordinarios de manera instantania.
